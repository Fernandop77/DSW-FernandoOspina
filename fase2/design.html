Propósito

El proposito fundamental por la cual que se creo este sitio es para diversificar y fundamentar las actitudes de los temas referentes sobre las Etiquetas Básicas de HTML, atributos y valores que se manejan en este.

Etiquetas basicas de HTML

En HTML las etiquetas sirven como la estructura elemental de este, estas se componen en su mayoría por otras propiedades las cuales son los valores y los atributos los cuales ayudan a formar una base la cual servirá para ir realizando un proyecto de manera estructurada y organizada. Entre estas etiquetas tenemos unas en concreto que son básicas para crear cualquier documento HTML y servirán para dar una base estructurada y fundamental para la creación de proyectos y diversos manejos para la elaboración de tus trabajos, por lo tanto se expondrán las mas importantes y su función en nuestro código.
body
Indica la parte del cuerpo del contenido de un documento HTML. Es una etiqueta esencial para cualquier documento ya que indica donde empieza el contenido visible del documento.
head
La parte superior del documento HTML, es donde podremos indicar los metadatos: título del documento, hojas de estilos, javaScript, CSS.
div
Un elemento que es usado mayoritariamente para agrupar otros elementos y actuar como plantilla de otros controles. La etiqueta div nos ayuda a estructurar el documento en secciones.
a
Es una etiqueta que nos ayuda a poder crear un enlace a una página web. El atributo principal de la etiqueta HTML es href, donde pondremos el enlace al que queremos conectar. Otro atributo muy usado es target, el cual nos sirve para indicar si el enlace se abrirá en una nueva ventana o en la misma.
strong
Si tienes mucho texto, es importante poder dar énfasis a una parte en concreto, con la etiqueta strong lo podemos hacer.
br
Con esta etiqueta HTML le podemos decir al navegador que viene un salto de línea. Nos sirve para hacer el texto más leíble.
h1-h6
Hay diferentes niveles de títulos, del 1 al 6. Las etiquetas H + número nos permiten indicar la importancia del título y para estructurar el contenido, de esta forma ayudamos a los bots a entender la importancia del contenido.
img
Usamos la etiqueta IMG para mostrar imágenes dentro del contenido. Necesita el atributa src para funcionar, ya que será donde indicaremos desde donde tiene que mostrar la imagen.
ol-li-ul
Las etiquetas OL y LI nos sirven para crear listas, OL para listas ordenadas y UL para listas sin orden, dentro de las listas, los elementos se identifican con la etiqueta LI.
p
Etiqueta que nos sirve para agrupar texto dentro de un parágrafo. El propósito es poder hacer el contenido más fácil de leer y organizado.
span
Con la etiqueta podemos personalizar el estilo de solamente una parte del texto.(Idevik, 2021)

Atributos y valores

Los elementos en HTML tienen una serie de atributos los cuales son valores adicionales que se encargan de configurar y ajustan de manera practica para el usuario para que estos obtengan la configuración deseada y que esta cumpla con sus criterios previos. Los atributos son variados y constan de diversas funciones según su tipo estos se dividen entre 4 grupos los cuales son:
Atributos básicos
Estos son básicos y se disponen en su mayoría en las etiquetas HTML algunos para exponer serian:
Id
El cual establece un indicador único a cada elemento.
class
El cual sirve para aplicar la clase a diversos estilos al elemento.
Style
Sirve para aplicar de forma directa los estíos a un elemento.
title
Sirve para establecer un título al elemento.
Atributos de Internacionalización
Este tipo de atributos sirve para ciertas paginas que desean darse exposición a diversas lenguas alrededor del mundo y necesitan de disponer de variados idiomas en su contenido, entre estos tenemos:
lang
El cual sirve para indicar el idioma deseado de configuración.
dir
Sirve para indicar la dirección que tiene el texto.
Atributos de eventos
Estos se utilizan en paginas web que contengan en su manejo Javascript los cuales servirán para hacerla más dinámica en su contenido.
Atributos de foco
Sirven para determinar que un elemento que deja de estar seleccionado deje de tener el foco y en su cambio se disponga de otro en el que se esté navegando, entre estos tenemos: Accesskey
El cual sirve para predeterminar una tecla de acceso rápido a cierto elemento HTML.
